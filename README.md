#  E-Commerce Website

A full-stack **E-Commerce Web Application** developed using **HTML, CSS, JavaScript, PHP, and MySQL**. The project provides an online shopping platform where users can browse products, add items to their cart, and place orders, while administrators can manage products and users through an admin panel.

---

##  Features

###  User Module

* User Registration and Login
* Secure Authentication
* Browse Products
* Search Products
* View Product Details
* Add to Cart
* Update Cart Quantity
* Remove Items from Cart
* Place Orders

###  Admin Module

* Admin Login
* Add New Products
* Edit Existing Products
* Delete Products
* Manage Product Inventory
* View Customer Orders
* Manage Users

##  Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/E-commarce-project.git
```

2. Move the project folder to the `htdocs` directory (XAMPP).

3. Start **Apache** and **MySQL** from the XAMPP Control Panel.

4. Open **phpMyAdmin** and create a database.

5. Import the provided `ecommerce.sql` file.

6. Update database credentials in the configuration file if required.

7. Open the project in your browser:

```
http://localhost/ecommerce-project/
```
